Project robot vacuum cleaner

The goal of this project was to develop a program to be implemented on a robot that will simulate the operation of a robot vacuum cleaner.
This program is stored on the embedded system that is this robot, and has two functions,
the first is to make a map of his environment by moving in it
the second is to trace its course in order to know where it went
This project lasted 1 month and a half during the year 2021, I worked on it alone

Related Skills

  • C++ procedural development
  • Use of 2D drawing libraries
  • Raspbian linux environment
  • Use of embedded modules

My project outcomes:

  • Mastery of procedural C++
  • Robot moving by mapping its environment and representing its path
