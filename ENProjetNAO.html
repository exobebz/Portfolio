The Nao project

This project was carried out for 6 months in 2022, as a team with two technicians, our goal was to use a humanoid robot called NAO,
as well as a touch tablet to make animations with residents in an EHPAD.

Related Skills

  • Project Management
  • Android Development
  • Definition of specifications
  • Interface creation

My project outcomes:

    Create an Android application usable by EHPAD staff with the following requirements:

  • Make the robot dance, talk and move
  • Have a simple and intuitive interface
  • Use Wi-Fi to communicate between the tablet and the robot
