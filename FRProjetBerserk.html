Le projet Berserk

Ce projet est personnel, avec deux amis et anciens camarades de promotion de BTS nous avons décidé de nous alncer dans le développement d'un jeu vidéo,
ce projet va nous permettre a chacun de développer les compétences qui nous paraîssent nous correspondre le mieux.
En effet chacun d'entre nous a une partie bien spécifique du projet à gérer.

Mes tâches lors de ce projet :

  • Gérer l'avancement du projet, les deadlines
  • Donner la direction du projet
  • Définir les besoins et aboutissant

Quelques informations sur ce projet

  • Le jeu est un RPG/beat them all dans un univers médieval DarkFanstasy
  • Une beta a été codé en c++, la reprise du projet après une grosse pause modifiera cette base, il sera développé sur Unity.
  • Les trois personnes travaillant sur le projet sont trois étudiants dans des cursus différents
